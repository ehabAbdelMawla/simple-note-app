## Getting Started

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/ehabAbdelMawla/simple-note-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3 Install NPM packages
   ```sh
   npm start
   ```
